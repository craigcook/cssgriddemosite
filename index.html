Mozilla on CSS Grid

CSS Grid

CSS Grid is a new toolset built directly into web browsers. Grid makes it possible to do innovative page layouts using CSS, without needing heavy frameworks. Grid is a layout framework.

CSS Grid shipped in Firefox 52, on March 7, 2017.

It's coming soon to Chrome 57, on March 14. And to Safari 10.1, in late March or early April.

CSS Grid makes it easy to code the layouts we see most. And it makes it possible to / bringing the power and traditiona of graphic design to the web.

This is not Bootstrap. This is not HTML Tables done in CSS. Don't assume you know what this is. You've never seen anything like it before. Let's take a look, together.
