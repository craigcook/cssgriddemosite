Mozilla on CSS Grid

CSS Grid

CSS Grid is a new toolset built directly into web browsers. Grid makes it possible to do innovative page layouts using CSS, without needing heavy frameworks. Grid is a layout framework.

CSS Grid shipped in Firefox 52, on March 7, 2017.

It's coming soon to Chrome 57, on March 14. And to Safari 10.1, in late March or early April.

Example 1

Here's the heart of the code running this example. The images are part of an HTML list.

<ul class="gallery">
  <li><img src="image1.jpg"></li>
  <li><img src="image2.jpg"></li>
  <li><img src="image3.jpg"></li>
</ul>

The layout is accomplished with incredibly simple CSS. Three lines total accomplish the entire layout.

ul.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
  grid-gap: 1rem;
}

How is this possible? CSS Grid automatically places all of the direct children of a grid container, putting each one in the next cell on the grid. It takes the first list item and puts it in the first box. Then it takes the second item and puts it in the second box. Magic. This is called the Grid Autoplacement Algorithm. It makes coding the layouts we've been designing for years much easier.

Let's take a deeper look at the code, by using Firefox's Grid Inspector tool to see what's going on.

  1. [screenshot] Right-click on part of the example to open the Developer Tools.
  2. [screenshot] You can see the HTML markup on the left, and the CSS on the right. Now click on <ul class="gallery"> to see the CSS that has been applied to the grid container.
  3. [screenshot] Find where it says display: grid;, and click on the tiny hash icon next to the word grid.
  4. [screenshot] That turns on our Grid Inspector tool, letting you see the lines for of the grid.
  5. Now play around. You can redefine the grid. Try changing the minmax value from (100, 1fr) to (200, 1fr) and see what happens. Change the repeat value to a number, resize the browser window, and see what's different. Change the value of the grid-gap. You can mess with the design right here in the browser, experimenting and learning about the new possibilities.
